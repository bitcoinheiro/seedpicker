Want to create your own 24 word Seed Phrase for your Bitcoin wallet? You can select the first 23 from the official list of 2048 words, but the 24th word must be calculated. This is what SeedPicker can help you with.
You also get the corresponding Extended Public Key, in Zpub (mainnet) or Vpub (testnet) format. This will enable you to create multisig wallets without needing a hardware wallet.
Make sure your 23 words are generated in a random and secure fashion! Read the guide for how to accomplish this.
Key formats and derivation paths have been selected to work well with Electrum, but may also work with other wallets.

Warning

This page is designed to be used on an offline and fully secure computer.
You can try it out in online mode, but before using it for amounts you are not ready to lose, please read and follow all the instructions here.
