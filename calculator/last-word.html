This tool enables you to create the Extended Public Keys (in Zpub or Vpub format) needed to create Electrum multisig wallets, without needing a hardware wallet.
You can bring your own 23 random words, but make sure these words are generated in a random and secure fashion! Read the guide for how to accomplish this.
Key formats and derivation paths have been selected in order to work well together with Electrum, but may also work with other wallets.
WARNING: Before using this calculator, read all instructions here.
WARNING: Only use this calculator on an offline and fully secured computer! (see instructions)
